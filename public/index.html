⚙ 模型目录配置
模型列表 JSON URL
模型列表 API 地址。Docker 部署默认使用内置 /api/models。如果用静态 JSON 文件，改为文件路径（如 ./models.json）。
模型文件基础路径
Docker 部署时留空即可（API 返回的 URL 已是完整路径）。静态部署时填入模型文件所在的 Web 路径前缀。
💡 部署提示
Docker 部署：把模型放入 models/ 目录，执行 docker compose up -d，无需额外配置。

静态部署：运行 python3 scan.py /模型目录 -u /models/ 生成 JSON，再将页面与 JSON 部署到同一 Web 目录。
